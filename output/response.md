This project, **Voice-enabled-banking-system-**, is a voice-controlled personal banking assistant. It allows a user to perform banking tasks—such as making payments, checking transaction history, and retrieving account details—using natural language voice commands.\n\n### Key Components:\n*   **Voice Authentication:** The system uses speaker verification (via the `speechbrain` library) to ensure that only the enrolled user can access the banking assistant. A separate script (`user_voice.py`) is used to record and save an initial voice profile.\n*   **Agentic Framework:** It employs a Large Language Model (LLM) agent powered by LangChain to interpret voice commands and call specific tools to interact with the user's data.\n*   **Banking Tools:** The assistant is equipped with the following tools defined in `agent_tools.py`:\n    *   `make_payment`: Handles money transfers by verifying recipient availability, checking account balance, and updating the transaction history.\n    *   `get_payment_history`: Retrieves past transactions for analytical queries.\n    *   `get_user_details`: Provides information about the user’s account, such as balance and contact lists.\n*   **Voice Processing:**\n    *   **Speech-to-Text (STT):** Uses Google's speech recognition to transcribe the user's voice into text.\n    *   **Text-to-Speech (TTS):** Uses `pyttsx3` to provide audible, conversational responses to the user.\n\nIn short, the project acts as a secure, voice-activated bridge between the user and their financial data, using an AI agent to handle transactional logic while managing voice input/output locally.
