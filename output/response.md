The AI-Lost-and-Found project is a smart application designed to help users find lost items or report found ones. It leverages a robust ML pipeline including Google Vision API for image analysis, Groq AI for detailed descriptions, and Sentence Transformers for creating embedding vectors. The core functionality relies on a smart matching algorithm that uses cosine similarity to identify potential matches between lost and found items. The application features secure Firebase Authentication, image upload capabilities, and integrates with Firebase Storage for images and Firestore for metadata. The backend is built with FastAPI and Python, while the frontend uses HTML, CSS, and JavaScript. The matching process, with a similarity threshold of 0.80, runs on the backend and returns all matches without deleting stored items. Future enhancements include integrating a vector database for scalable search, adjustable similarity thresholds, and user notifications.
