**Step 1: Expense Logging**
*   **Done:** A user can input an expense amount, select a category (e.g., "Food Delivery," "Subscriptions," "Groceries," "Utilities"), and add a brief note. The expense is saved with a timestamp.

**Step 2: Monthly Summary View**
*   **Done:** The application displays a summary of total spending for the current month, broken down by category. Each category shows the total amount spent within it for the month.

**Step 3: Data Persistence**
*   **Done:** All entered expenses are stored so they are available when the application is closed and reopened.
